# UTO Frontend (Next.js + Tailwind)

## Quickstart
1. Install deps: `npm install`
2. Dev: `npm run dev` (http://localhost:3000)
3. Build: `npm run build` and `npm start`

## Env vars
- `NEXT_PUBLIC_API_URL` - URL to the backend API (default `/api` for local demo)

## Deploy to Vercel
